# 3D AI 数字艺术主页设计方案

## 设计参考

- **左侧参考（eppright homes）**：深色导航栏、大图 Hero、金色强调色、视频播放元素
- **右侧参考（Better Off®）**：极简主义、大胆黑色文字、高对比度、简洁布局

## 实现步骤

### 1. 页面结构设计

在 `src/views/Home.vue` 中实现以下区块：

- **顶部导航栏**：深色背景，包含品牌 Logo（3D/AI 图标）、导航链接（首页、作品展示、服务、关于我们、联系我们）、CTA 按钮
- **Hero 区域**：大尺寸 3D/AI 数字艺术作品背景图，叠加大号标题文字（如"创造无限可能"），副标题描述，视频播放按钮或 CTA
- **服务介绍区块**：卡片式布局展示核心服务（3D建模、AI创作、数字艺术等）
- **作品展示区**：网格布局展示精选作品，支持轮播或分页
- **价值主张区块**：大号文字展示核心理念（参考 Better Off® 风格）
- **关于我们**：团队介绍、使命愿景
- **底部联系信息**

### 2. 视觉元素

- **配色方案**：深色背景（深灰/黑色）+ 亮色强调（金色/蓝色/紫色渐变）
- **字体**：大号粗体标题，清晰易读的正文
- **图片资源**：使用 3D、AI、数字艺术相关图片（可从现有 assets 或占位图开始）
- **毛玻璃效果**：为图片卡片添加 `backdrop-filter: blur()` 和半透明背景，营造现代玻璃态视觉效果
- **3D 沉浸感**：Hero 模块加入透视网格、星云光效与悬浮信息卡片，并结合鼠标移动驱动 3D 倾斜与浮动动画

### 3. 交互功能

- 导航栏滚动效果
- Hero 区域视频播放（可选）
- 指针驱动 Hero 3D 倾斜与浮动信息卡片动画
- 作品展示轮播/网格切换
- 平滑滚动动画

### 4. 响应式适配

- 移动端优化布局
- 平板和桌面端适配

## 技术实现

- 保持 Vue 3 + TypeScript 架构
- 使用 Element Plus 组件库
- SCSS 样式，支持深色主题
- 响应式设计（移动端优先）
